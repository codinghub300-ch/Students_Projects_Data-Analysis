#  Project Overview

The Titanic Passenger Analysis Dashboard provides interactive data analysis and visualization of the famous Titanic passenger dataset. 
The application enables users to explore passenger information, analyze survival statistics, perform NumPy and GroupBy analysis, filter passengers based on multiple criteria, and visualize insights through interactive charts built with Streamlit.

###  Features

-  Dataset Preview
-  Statistical Analysis
-  NumPy Operations
-  GroupBy Analysis
-  Passenger Filtering
-  Interactive Charts
-  Download Visualizations
